Portal Quickstarts

The quickstarts demonstrate several ways how to create Portlets for GateIn Portal. They provide small, specific, working examples that can be used as a reference for your own project.

Compatibility Information

This release of Portal Quickstarts is compatible with GateIn Portal 3.6 running on JBoss AS.

Available Quickstarts

  • Simplest Hello World Portlet
  • JSF2 Hello World Portlet
  • JSF2+RF4 Hello World Portlet
  • Portal Extension
  • Sample Portal
